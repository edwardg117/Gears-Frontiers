# Block Tags for Diagonal Fences Datapack

This directory contains custom block tags used in the Diagonal Fences datapack. Tags in here are read by the Diagonal Fences mod to determine which fences should have diagonal connections.

## Tag Files

### non_diagonal_fences.json

This tag contains block IDs that should not have diagonal connections. The fences listed in this tag will not connect diagonally to other fences. The fences included in this tag are:

- Macaw's grass topped walls
    - The wall got stuck in the cross configuration [Bug on Diag side](https://github.com/Fuzss/diagonalfences/issues/25) - [Bug Macaw's side](https://github.com/sketchmacaw/MacawsModsIssues/issues/509). May get fixed at some point
- Macaw's wired fences 
    - Mobs don't get damaged like they should
