# Entity Type Tags for Gears & Frontiers Datapack

This directory contains custom entity type tags used in the Gears & Frontiers datapack. Tags in here can be read by Carry On and used to modify what can and can't be picked up.

## Tag Files

### capturing_not_supported.json

This tag contains entity types that are not supported for capturing. It is used to prevent players from picking up entities with Carry On. The entities included in this tag are:

- Astikor Carts Redux carts (plow, supply cart, hand cart, reaper, animal cart, seed drill)
